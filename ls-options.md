# ls 命令选项速查表

| 选项 | 参数 | 说明 |
|:---|:---|:---|
| `-A`, `--almost-all` | — | 列出除 `.` 和 `..` 以外的所有条目 |
| `--author` | — | 与 `-l` 配合使用，打印每个文件的作者 |
| `-b`, `--escape` | — | 对非图形字符显示 C 风格的转义序列 |
| `--block-size=SIZE` | `SIZE` | 与 `-l` 配合使用，按指定大小缩放并打印尺寸；例如 `--block-size=M`；详见下方 SIZE 格式说明 |
| `-B`, `--ignore-backups` | — | 不列出以 `~` 结尾的隐含条目 |
| `-c` | — | 与 `-lt` 配合：按 ctime（文件状态信息最后修改时间）排序并显示；与 `-l` 配合：显示 ctime 并按名称排序；其他情况：按 ctime 排序，最新的排前面 |
| `-C` | — | 按列列出条目 |
| `--color[=WHEN]` | `WHEN` | 对输出进行着色；`WHEN` 可选值：`always`（默认）、`auto`、`never`；详见下方说明 |
| `-d`, `--directory` | — | 列出目录本身，而非目录中的内容 |
| `-D`, `--dired` | — | 生成适用于 Emacs dired 模式的输出 |
