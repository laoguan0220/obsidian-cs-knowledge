## 1. 左边这串是什么？

plaintext

```
root@localhost:/mnt/c/Users/13411#
```

这是 **Linux 终端提示符**，告诉你当前状态：

- `root`：你现在是 **超级管理员（最高权限）**
- `localhost`：本机
- `/mnt/c/Users/13411`：**当前所在文件夹**
    
    - 这是 **Windows 的用户文件夹**
    
- `#`：管理员提示符

---

## 2. 右边命令 `cd /xv6-RISC-V` 是什么意思？

### **cd = Change Directory → 切换文件夹**

### 完整含义：

**立刻从当前文件夹，切换到系统根目录下的 /xv6-RISC-V 文件夹**

---

## 3. 路径 `/xv6-RISC-V` 是什么？

- `/` = **Linux 系统根目录**
- `/xv6-RISC-V` = 根目录下面的 **xv6 项目文件夹**

  关于 CapsLock / Esc 互换

  WSL 里的键盘输入由 Windows 管理，在 WSL 层面改不了。你目前有两个方案：

  - 随身方案（推荐）：已经配好了 jj / jk 代替 Esc，在任何机器上都生效
  - 物理互换方案：安装 https://github.com/microsoft/PowerToys → Keyboard Manager → Remap a key，把 Caps Lock 映射到
  Esc。这会影响整个 Windows 系统，不只是在
   vim 里

  要生效的话，在 WSL 里执行 source ~/.bashrc 或者新开一个终端即可。
