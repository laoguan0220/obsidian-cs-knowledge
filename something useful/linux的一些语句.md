## 1. 左边这串是什么？

plaintext

```
root@localhost:/mnt/c/Users/13411#
```

这是 **Linux 终端提示符**，告诉你当前状态：

- `root`：你现在是 **超级管理员（最高权限）**
- `localhost`：本机
- `/mnt/c/Users/13411`：**当前所在文件夹**
    
    - 这是 **Windows 的用户文件夹**
    
- `#`：管理员提示符

---

## 2. 右边命令 `cd /xv6-RISC-V` 是什么意思？

### **cd = Change Directory → 切换文件夹**

### 完整含义：

**立刻从当前文件夹，切换到系统根目录下的 /xv6-RISC-V 文件夹**

---

## 3. 路径 `/xv6-RISC-V` 是什么？

- `/` = **Linux 系统根目录**
- `/xv6-RISC-V` = 根目录下面的 **xv6 项目文件夹**
