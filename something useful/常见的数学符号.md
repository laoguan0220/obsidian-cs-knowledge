| 符号类型         | 显示效果    | LaTeX 语法              | 适用场景               |
| ------------ | ------- | --------------------- | ------------------ |
| **基础运算**     | +       | `$+$`                 | 加法、增量              |
|              | −       | `$-$`                 | 减法、减量              |
|              | ×       | `$\times$`            | 乘法（避免与 x 混淆）       |
|              | ÷       | `$\div$`              | 除法                 |
|              | =       | `$=$`                 | 等于                 |
|              | =      | `$\neq$`              | 不等于                |
|              | ≈       | `$\approx$`           | 约等于                |
|              | >       | `$>$`                 | 大于                 |
|              | <       | `$<$`                 | 小于                 |
|              | ≥       | `$\ge$`               | 大于等于（≥同效）          |
|              | ≤       | `$\le$`               | 小于等于（≤同效）          |
|              | ±       | `$\pm$`               | 加减号                |
|              | ∑       | `$\sum$`              | 求和（基础）             |
|              | ∑i=1n​  | `$\sum_{i=1}^n$`      | 定范围求和（树状数组 / 前缀和）  |
|              | ∏       | `$\prod$`             | 求积                 |
|              | ∏i=1n​  | `$\prod_{i=1}^n$`     | 定范围求积              |
| **集合逻辑**     | ∈       | `$\in$`               | 属于（如i∈[1,n]）       |
|              | ∈/      | `$\notin$`            | 不属于                |
|              | ⊂       | `$\subset$`           | 真子集                |
|              | ⊆       | `$\subseteq$`         | 子集                 |
|              | ∪       | `$\cup$`              | 并集                 |
|              | ∩       | `$\cap$`              | 交集                 |
|              | ∅       | `$\emptyset$`         | 空集                 |
|              | ∀       | `$\forall$`           | 任意（如∀i∈N）          |
|              | ∃       | `$\exists$`           | 存在                 |
|              | ⇒       | `$\Rightarrow$`       | 推出                 |
|              | ⇔       | `$\Leftrightarrow$`   | 等价                 |
| **上下标 / 根号** | an​     | `$a_n$`               | 下标（数组元素ai​）        |
|              | an+1​   | `$a_{n+1}$`           | 多字符下标（需加 {}）       |
|              | a2      | `$a^2$`               | 上标（平方 / 次方）        |
|              | an+1    | `$a^{n+1}$`           | 多字符上标（需加 {}）       |
|              | x​      | `$\sqrt{x}$`          | 平方根                |
|              | nx​     | `$\sqrt[n]{x}$`       | n 次方根              |
|              | logn    | `$\log n$`            | 对数（无底数）            |
|              | log2​n  | `$\log_2 n$`          | 2 为底对数（算法时间复杂度）    |
|              | lnn     | `$\ln n$`             | 自然对数               |
| **极限 / 积分**  | lim     | `$\lim$`              | 极限（基础）             |
|              | limn→∞​ | `$\lim_{n\to\infty}$` | 趋于无穷极限             |
|              | ∫       | `$\int$`              | 不定积分               |
|              | ∫ab​    | `$\int_a^b$`          | 定积分（a 到 b）         |
|              | ∞       | `$\infty$`            | 无穷大（正）             |
|              | −∞      | `$-\infty$`           | 无穷大（负）             |
| **离散数学**     | \mod    | `$\mod$`              | 取模（如xmodn）         |
|              | gcd     | `$\gcd$`              | 最大公约数（gcd(a,b)）    |
|              | lcm     | `$\text{lcm}$`        | 最小公倍数（lcm(a,b)）    |
|              | ∣       | `$\mid$`              | 整除（如n∣m）           |
|              | ∤       | `$\nmid$`             | 不整除                |
|              | (kn​)   | `$\binom{n}{k}$`      | 组合数（n 选 k）         |
| **特殊符号**     | π       | `$\pi$`               | 圆周率                |
|              | e       | `$e$`                 | 自然常数               |
|              | Δ       | `$\Delta$`            | 增量 / 变化量（如Δx）      |
|              | θ       | `$\theta$`            | 复杂度符号（θ(n)）        |
|              | O(n)    | `$O(n)$`              | 大 O 复杂度（算法时间 / 空间） |
|              | Ω(n)    | `$\Omega(n)$`         | 下界复杂度              |
|              | Θ(n)    | `$\Theta(n)$`         | 紧界复杂度              |
|              | …       | `$\dots$`             | 省略号（水平）            |
|              | ⋮       | `$\vdots$`            | 省略号（垂直）            |
|              | ⋯       | `$\dotsb$`            | 省略号（二进制 / 序列）      |
