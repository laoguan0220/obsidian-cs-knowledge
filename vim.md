# Vimtutor 全部必背指令「极简表格版」

只留**必考、必用、写代码高频**，无废话，直接背。

> 默认：**普通模式**生效，冒号命令需先按 `Esc`

## 一、模式切换（最核心）


表格

|指令|作用|
|---|---|
|`i`|光标前插入|
|`a`|光标后插入|
|`o`|下方新建空行并插入|
|`I`|行首插入|
|`A`|行尾插入|
|`O`|上方新建空行并插入|
|`Esc`|退回普通模式|

## 二、光标移动

表格

|指令|作用|
|---|---|
|`h`|左移|
|`j`|下移|
|`k`|上移|
|`l`|右移|
|`$`|跳到行尾|
|`0`|跳到行首|
|`gg`|文件开头|
|`G`|文件末尾|

## 三、删除 / 修改

表格

|指令|作用|
|---|---|
|`x`|删除光标单个字符|
|`dd`|删除 / 剪切整行|
|`数字+dd`|删除指定行数（如`3dd`）|
|`r+字符`|快速替换单个字符|
|`cw`|删除当前单词并进入编辑|

## 四、复制 / 粘贴

表格

|指令|作用|
|---|---|
|`yy`|复制整行|
|`数字+yy`|复制多行|
|`p`|向下 / 向右粘贴|

## 五、撤销 / 重做

表格

|指令|作用|
|---|---|
|`u`|撤销上一步|
|`U`|撤销整行修改|
|`Ctrl + r`|重做（恢复撤销）|

## 六、保存 & 退出（命令模式 `:`）

表格

|指令|作用|
|---|---|
|`:w`|只保存，不退出|
|`:wq`|保存并退出|
|`:q`|正常退出（无修改）|
|`:q!`|不保存，强制退出|

## 七、搜索 & 替换

表格

|指令|作用|
|---|---|
|`/xxx`|向下搜索 xxx|
|`n`|下一个匹配项|
|`N`|上一个匹配项|
|`%`|跳转匹配括号 `() [] {}`|
|`:s/old/new`|本行替换第一个|
|`:s/old/new/g`|本行全部替换|
|`:%s/old/new/g`|全文全部替换|

## 八、可视化选中

表格

|指令|作用|
|---|---|
|`v`|字符选择|
|`V`|整行选择|
|`Ctrl+v`|矩形块选择|

## 九、分屏窗口操作

表格

| 指令           | 作用     |
| ------------ | ------ |
| `:sp`        | 水平分屏   |
| `:vsp`       | 垂直分屏   |
| `Ctrl+w w`   | 切换分屏窗口 |
| `Ctrl+w h/l` | 左右窗口切换 |
